# Blue2
A directory of scripts for every function from home automation to web-scraping. 

# Structure
Some scripts will attempt to pull or save information to certain directories. The expected structure is shown below:
 - `~/` (home)
  - `/scripts`
   - `/sensors`
    - `/temp`
    - `/light`
    - `/sound`
   - `/global`
    - `/main`
    - `/pylogger`
   - `/weather`
   - `/camera`
  - `/keys`
  - `/data`
  - `/logs`
